This folder contains scripts used for applying metaCCA. Please read the "MetaCCA Implementation" guide to understand what each of these files does and in what order they were executed.

All of the data was stored under the data directory and the map of the directory was as follows:

Data 
      -- plink_data: directory to store hg19 gene data 

      -- g1000p_EUR: directory to store the 1000genome data for the European population

		      -- prune: directory to store the ld pruning data
	 
      -- raw: directory to store raw GWAS files

      -- processed: directory to store all the processed files

The results are saved in the tab-delimited file: metcca_results_pBonf_sig.csv
